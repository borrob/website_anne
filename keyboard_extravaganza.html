Keyboard Extravaganza

Keyboard Extravaganza is a flexible program that explores music for various combinations of keyboard instruments through the centuries.The concert features duo works for piano, harpsichord, toypianos, organ, celesta and/or harmonium. Whilst some of these instruments are well known to many, the unusual combinations create unique and exciting musical experiences!
